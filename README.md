Photography Gallery Generator
===================


This is a simple bash script that parses a folder of images (default: jpg), processes them (reduce, resize, and applies a watermark) using ImageMagick, and generates an HTML index for use in a photography website.

See below for an example of both a processed image file and the resulting HTML index:
